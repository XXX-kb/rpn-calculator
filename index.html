RPN 计算器使用手册

安装和编译指南

🌐 GitHub Codespace (推荐)

  1. 在项目页面点击 Code → Codespaces → Create codespace
  2. 在终端中执行:
# 编译
g++ rpn_calc.cpp -o rpn_calc
# 运行
./rpn_calc

🪟 Windows 系统

使用 Visual C++ 编译器:

# 编译"
cl /EHsc rpn_calc.cpp
# 运行
rpn_calc.exe

需要安装 Visual Studio 并包含 C++ 开发工具

🐧 Linux 系统

   # 编译
   g++ rpn_calc.cpp -o rpn_calc
   # 运行
   ./rpn_calc

基本使用示例

输入：5 2 +

结果：7

输入：3 4 * 2 /

结果：6

输入：5 5 + ;10 2 / 输出： 批量计算 2 个表达式：

1. 5 5 + = 10.000000

2. 10 2 / = 5.000000

支持的操作

  • 加法: +
  • 减法: -
  • 乘法: *
  • 除法: /
